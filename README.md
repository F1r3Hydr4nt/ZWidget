# ZWidget
Ze-Ultra-Widget for Android Tidiness & Travel

## Features
1. Main Location Time with weather (temperature and emoji)
2. Sub-selectable list of location time display (click-through to briefly display current weather description)
3. A textfield local currency to home currency converter without exiting widget (using xe.com)
4. Bitcoin-SV current trading price
5. Icon Button shortcuts to Google Translate, Maps & Lens

## Tech Stack
- Kotlin
- Android Jetpack (Compose for Glance widgets)
- Retrofit for API calls
- WorkManager for periodic updates
